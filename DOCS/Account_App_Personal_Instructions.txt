SCHWAB ONE ACCOUNT APPLICATION - ADVISOR FIELD GUIDE
====

STANDARD VALUES
---------------
All checkboxes: /Yes (checked), /Off (unchecked)
All dates: mm/dd/yyyy format
Print in CAPITAL LETTERS

IMPORTANT RULES
---------------
- LEAVE ALL FIELDS BLANK THAT ARE NOT NEEDED
- Don't make up any details
- If uncertain, ASK user before attempting
- Margin not available for Custodial, Conservatorship, Guardianship or Estate accounts
- Estate/Guardianship/Conservatorship require court order certified within last 12 months

====
